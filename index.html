Giới Thiệu về Podman

Podman là một công cụ quản lý container mở rộng, hỗ trợ phát triển, quản lý và triển khai ứng dụng trong môi trường container hóa. Nó là một phần của dự án podman.io và được xây dựng để làm việc với Linux Containers.

Cài đặt Podman

Để cài đặt Podman trên hệ điều hành của bạn, bạn có thể sử dụng lệnh sau tùy theo bản phân phối Linux của bạn:

sudo apt-get install -y podman (cho Ubuntu và Debian)
sudo yum install -y podman (cho Fedora, CentOS)

Lệnh cơ bản

Một số lệnh cơ bản để bắt đầu sử dụng Podman:

  • podman run - Chạy một container mới.
  • podman ps - Hiển thị danh sách các container đang chạy.
  • podman images - Hiển thị danh sách các image.
